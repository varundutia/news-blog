# 📰 When Search Stops Sending Traffic

An interactive digital storytelling experience exploring how AI-powered search summaries are reshaping the web, changing user behaviour, and impacting publishers, creators, and online businesses.

Built as part of a Digital Storytelling project at **University College Dublin**, the website combines narrative, visual design, and interactive web development to communicate a contemporary issue in technology.

---

# 🌐 Live Demo

👉 https://varundutia.github.io/news-blog/

---

# 📖 Overview

Search engines have traditionally acted as gateways to the web by directing users to publisher websites. The emergence of AI-generated search summaries is changing this relationship by allowing users to receive complete answers without visiting the original source.

This project presents this transformation through an interactive storytelling experience that combines research, narrative writing, and modern web design.

Rather than presenting information as a traditional article, the website guides readers through the topic using scrolling interactions, visual hierarchy, and responsive layouts.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/varundutia/news-blog.git
```

Navigate into the project

```bash
cd news-blog
```

Run locally

```bash
python3 -m http.server
```

Then visit:

```
http://localhost:8000
```

Alternatively, simply open `index.html` in a web browser.
